# Story Adaptations

These are short in-universe treatments of the threads in [Story Threads](../STORY_THREADS.md).

Each one is written in a form that suits the material: testimony, machine log, oral tradition, bureaucratic memo, or conventional prose. They are not meant to be final collectible texts word-for-word, but they provide a stronger sense of scene, voice, and dramatic shape for future quest writing and environmental storytelling.
